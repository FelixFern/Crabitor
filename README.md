## How to Run ?
1. Git clone this repository with
   ```
   git clone https://github.com/FelixFern/Crabitor
   ```
2. Install all the needed dependencies with
   ```
   pnpm i
   ```
3. Run the project and enjoy
   ```
   pnpm tauri dev
   ```
